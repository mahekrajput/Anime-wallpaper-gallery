# Anime-wallpaper-gallery
a front end project

A simple, responsive web gallery to browse high-quality anime wallpapers from Naruto, One Piece, and Death Note. Users can view wallpapers by series in a clean, easy-to-use layout.

# How It Works

Dark, cool UI.

Navigate between Home, Gallery, and About pages.

Gallery displays wallpapers categorized by series, loaded directly from online URLs.

Simple animations enhance the visual experience.

Geolocation button on the homepage shows your latitude and longitude.

# Tech Stack

HTML5 – Website structure

CSS3 – Styling and hover effects

JavaScript – Geolocation and interactive features

# Usage

Open index.html in a browser(preferably chrome or edge).

Go to Gallery to explore wallpapers by series.

Use the geolocation button on the homepage to see your current location.

# Notes

All images are loaded from online URLs — no local files needed.

Designed to be simple, responsive, and visually clean.
